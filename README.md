# Arkanoid JavaScript Remake

Classic arcade game remake built with JavaScript and HTML5 Canvas.
Focused on gameplay systems, collision detection and interactive browser-based mechanics.

## Live Demo

https://arkanoidjavascriptremake.vercel.app/

## Tech Stack

- JavaScript
- HTML5 Canvas
- HTML/CSS

## Features

- Collision detection system
- Ball physics and rebound logic
- Score tracking
- Arcade-style gameplay
- Responsive keyboard controls
- Browser-based game rendering

## Future Improvements

- React refactor for scalable component architecture
- Additional enemy and power-up systems
- Mobile support
- Sound effects and animations
